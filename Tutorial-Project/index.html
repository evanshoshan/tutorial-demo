Introduction

React is a front-end JavaScript library for building user interfaces. It was developed by Facebook to allow developers to create reusable UI components.

It is currently one of the most popular JavaScript libraries and has a strong foundation and large community behind it.

This tutorial will describe the basic knowledge needed to dive into React for the first time. I will presume that you have a basic understanding of HTML, CSS, and JavaScript. While React does not use HTML, it uses JSX which is very similar and has the same underlying principals.

Setup

Setting up a React project requires a great deal of configuration. For the purpose of this tutorial we will use something called create-react-app to create this application. This is a command line utility that allows you to get a react project setup and running with little configuration required.

Now in order to use this command line utility you must first install node.js onto your computer. This can be done by heading to nodejs.org and installing the most recent stable version. After it has been installed, you must install the create-react-app package onto your computer. Open a command prompt window and type:

npm install -g create-react-app

This installs the dependencies (globally) needed to use React and allows you to create your first application. Using the command prompt, navigate to the directory you want to create in and use the command:

create-react-app my-app

You have created your first React project....Congrats! Now lets dive in and start building the React demo app.

Instructions

As soon as you begin downloading dependencies, you will notice that there is a new folder in the directory you choose.
Once its done, your command prompt you should display instructions from React on how to begin.

Lets follow these instructions and see what happens. First, we will cd to the my-app. And then run npm start.

cd my-app
npm start

As you can see, it launches a live server, in this case localhost:3000 and displays a react splash screen telling you to edit src/App.js and reload.
